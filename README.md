# STSL (SeoulTech StackLands): 2024년 캡스톤디자인 프로젝트
STSL은 네덜란드의 게임 개발 스튜디오 [Sokpop Collective](https://sokpop.co/#home)에서 Unity 엔진을 사용하여 개발된 솔리테어, 건설 시뮬레이션 장르 게임 \<Stacklands\>를 Unreal 엔진으로 재구성하여 게임을 개발하는 프로젝트입니다.

## 역할 분담
- 프로그래밍: Timeshifts
- 기획, 리소스 제작: Minha77
